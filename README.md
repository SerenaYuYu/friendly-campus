# 🌿 友善校園正向環境教育管理系統

竹塘國小友善校園環境教育管理平台，供教師透過 LINE 登入使用，管理學生環境教育次數、登記髒亂區域、表揚自願清掃學生。

---

## ✨ 功能特色

### 🌍 公開頁面（無需登入）
- 查看全校整潔統計數字
- 瀏覽自願清掃表揚名單
- 對表揚學生按讚（家長、外部人士皆可）

### 👩‍🏫 教師功能
- **全校概況**：統計框可點擊查看學生明細（依權限顯示）
- **跑馬燈**：主畫面自動滾動顯示近期表揚學生
- **我的班級**：導師查看自己班學生環教狀況
- **登記髒亂區域**：支援匿名登記與拍照上傳
- **登記清掃**：選擇類型（環境教育／自願清掃）後登記學生
- **表揚名單**：查看表揚學生並按讚
- **意見回饋**：提交意見給管理員，支援匿名

### 🔑 管理員功能
- **全校各班環教統計**（完整班級明細）
- **學生管理**：新增、編輯學生資料，+/- 調整需完成次數
- **帳號管理**：審核教師權限申請、設定角色與班級
- **意見回覆**：查看並回覆老師意見

---

## 👥 教師申請流程

1. 老師用 LINE 開啟系統網址
2. 填寫姓名、職稱、班級送出申請
3. 管理員收到 **LINE 推播通知**，點連結進系統審核
4. 核准後老師自動進入完整系統，永久有效

---

## 🔒 權限設計

| 角色 | 各班統計 | 學生姓名 | 管理功能 |
|------|---------|---------|---------|
| 一般老師（無班級） | 全校總數 | ✗ | ✗ |
| 班級導師 | 自己班 | 自己班 | ✗ |
| 管理員 | 所有班級 | 全校 | ✅ |

## 🛠 技術架構

| 項目 | 使用技術 |
|------|----------|
| 前端 | HTML + Tailwind CSS + 純 JavaScript |
| 後端 | Firebase Firestore |
| 登入 | LINE LIFF（LINE 前端框架） |
| 部署 | Netlify |
| 通知 | Netlify Functions + LINE Messaging API |
