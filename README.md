# 🌿 友善校園正向環境教育管理系統

竹塘國小友善校園環境教育管理平台，供教師透過 LINE 登入使用，管理學生環境教育次數、登記髒亂區域、表揚自願清掃學生。

---

## ✨ 功能特色

### 👩‍🏫 教師功能
- **全校概況**：查看全校整潔統計數字（去識別化）
- **我的班級**：班級導師可查看自己班學生姓名與剩餘次數
- **登記髒亂區域**：任何老師皆可登記，支援匿名登記與拍照上傳
- **環教登記**：為學生登記環境教育服務，自動扣除剩餘次數
- **自願登記**：登記學生自願清掃，完成後列入表揚名單
- **表揚名單**：公開顯示自願清掃學生
- **意見回饋**：提交意見給管理員，支援匿名

### 🔑 管理員功能
- **學生管理**：新增、編輯學生資料與需完成次數
- **帳號管理**：審核教師權限申請、設定角色與班級
- **意見回覆**：查看並回覆老師意見

### 🌍 公開頁面
- 家長或外部人士可直接查看表揚名單與統計數字，無需登入

---

## 🔔 通知系統
- 老師申請帳號時，管理員自動收到 **LINE 推播通知**，附上系統連結直接審核

---

## 🛠 技術架構

| 項目 | 使用技術 |
|------|----------|
| 前端 | HTML + Tailwind CSS + 純 JavaScript |
| 後端 | Firebase Firestore |
| 登入 | LINE LIFF（LINE 前端框架） |
| 部署 | Netlify |
| 通知 | Netlify Functions + LINE Messaging API |

---

## 📱 使用方式

1. 老師透過 LINE 開啟 LIFF 網址
2. 第一次使用：填寫姓名、職稱、班級送出申請
3. 管理員在系統中審核並開通帳號
4. 開通後直接進入完整系統，無需再次設定

---

## 🔒 隱私保護

- 全校統計僅顯示數字，不顯示學生姓名
- 只有班級導師可查看自己班級學生資料
- 髒亂登記與意見回饋皆支援匿名
